EMediaPlayerPlugin

A plugin which integrates windows media player capabilities into eclipse. In addition it provides the option to share media files between the users in a local network.


Initial Setup:

1. After installing the plugin, click on media player icon which appears on the toolbar. It opens up a view named eMedia. Click on the drop down and select 'Preferences'.
2. Enter the name in the 'User' field to be visible in the network.
3. Set the local path to a folder which contains the media files on your computer.
4.Set the remote path to a shared folder on the network. If you don't wish to use the remote folder you can set the path to the same folder as the local path or you can choose another local folder which contains additional media files.

Playing Files:

To play a song click on the Add To Playlist from Playlist tab and double-click on the song to play it. Similarily to play a song from library just double-click the file.

Sharing Files:

Right click on the files in any of the sections, click on the share option to upload files to the shared network folder.

Using Shared URL's:

Right click on the files in any of the sections, select the 'Copy URL' and send the URL to other users in the network. Once the user copies the URL and clicks the 'Play URL' option from toolbar on the view, selected files in the URL will be automatically downloaded from repository if neccessary and will be added to the playlist.
Here is how a sample URL looks like when copied using the 'Copy Message' option from Skype '[4:51:18 PM] Sashikanth: [#eMediaSharedUrl]:#Sample Videos\Avengers.mp4'. The same can be played using the above option.

Playing Web URL:

Copy a valid URL(mp3,wmv etc.,) which can be played in the media player. Use the option 'Play URL' from toolbar to play the web URL.
Here is a sample sample link from IMDB Avengers Movie Trailor which can be played in the player.

Favourites:

To add song as a favourite click on the file and select Add To Favourites option. Favourites from different users can be viewed in the 'Favourites' section.

Exporting Files:

Right click on the files in any of the sections. Use the 'Export...' option to export all the files to a specified location. Files copied using this operation will be copied to respective folders in the target repository. To copy all the files into a single folder use 'Copy' option. Once the files are copied to clipboard you can use paste option to copy the files to the specified location.
